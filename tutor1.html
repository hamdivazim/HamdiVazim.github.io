'.popover' modifier for SwiftUI

Published 30/12/22

The popover modifier allows you to display a secondary view over the first. These appear as floating modals on iPadOS but fan out from the bottom on iOS.

All you need to display your own popover is a state to determine whether it is presented. Unlike alerts or action sheets, popovers can display anything you need it to! Just 'pop' in your state variable and what you want to display, and SwiftUI does the rest: 

        
struct ContentView: View {
    @State var showPopover = false
          
    var body: some View {
        Button("Show Menu") {
            showPopover = true
        }
          .popover(isPresented: $showPopover) {
              Text("Your content here")
          }
    }
}
